# Numerical Methods Course

This repository contains Python implementations of several numerical linear algebra algorithms developed during a Numerical Methods course.

The implementations are written in Python using Jupyter notebooks and focus on understanding the mathematical concepts behind the algorithms.

---

## Implemented Algorithms

- Classical Gram–Schmidt QR decomposition
- Modified Gram–Schmidt
- Singular Value Decomposition (SVD)
- Moore–Penrose pseudoinverse
- Gradient Descent optimization
- Linear Regression with K-Fold Cross Validation

---

## Purpose

The goal of this repository is educational: to implement and understand fundamental numerical linear algebra algorithms from scratch.

---

## Technologies

- Python  
- NumPy  
- Jupyter Notebook
